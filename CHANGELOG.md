## [2.1] - 2026-04-11

### 新增 / Added
- 📆 **workday() 工作日计算函数**：
  - 语法：`workday(起始日期, 工作日天数[, 额外节假日])`
  - 默认跳过周末，注释中列出跳过的节假日
  - 额外节假日用 `/` 分隔，`+` 添加（可省略），`-` 移除
  - 示例：`workday(Y20260411, 10, Y20260501/-Y20260412)`
- 📝 **智能日期/时间注释**：
  - 日期结果注释自动显示：X年X月X日 第XX周星期X
  - 时间结果注释自动显示：X时X分X秒
  - 日期差注释自动转换为：X年X月X周X天
  - 时间差注释自动转换为：X小时X分钟X秒
- 🔗 **hex() 支持嵌套函数调用**：如 `hex(swap(x))`

### 修复 / Fixed
- 🛠️ 修复 GUI 中 Ctrl+Enter 同时触发换行和计算的问题

### 改进 / Improved
- 📖 所有示例和帮助文档添加 swap/workday/hex(swap()) 用法
- 📚 更新全部文档

## [1.2] - 2026-03-13

### 新增 / Added
- ✨ **swap() 函数**：字节序交换函数
  - 自动检测字节数（1, 2, 4, 8）
  - 可用于表达式和赋值：`a = swap(b)`, `c = swap(d) * 2`
  - 可与 bitmap 配合：`bitmap(swap(x), 1)`
  - 常用于网络字节序转换
- 🔄 **撤销/恢复功能**（命令行版本）：
  - 命令行: 自动保存历史状态
  - 最多保存 50 个历史状态
  - API: `undo()`, `redo()`, `can_undo()`, `can_redo()`

### 改进 / Improved
- 🎨 **bitmap 显示优化**：
  - 保留16进制前导0（如 `0x0F` 而非 `0xF`）
  - 大端字节序：MSB（最高位）从索引 0 开始，从左到右 0→31
  - 小端字节序：LSB（最低位）从索引 0 开始，显示 31→0（从左到右）
  - 位值右对齐，与索引对齐
- 📖 更新帮助信息，添加 swap() 函数说明
- 🎯 优化表达式解析，支持函数嵌套

## [1.1] - 2026-03-05

### 重大变更 / Breaking Changes
- 🔄 **bitmap 语法重构**：将 `endian:` 命令和 `bitmap` 关键字合并为函数形式
  - 旧语法：`endian: big` + `bitmap 变量 = 值`
  - 新语法：`bitmap(值, 1)` 或 `变量 = bitmap(值, 1)`
  - `bitmap(值)` 或 `bitmap(值, 0)` 表示小端字节序（默认）
  - `bitmap(值, 1)` 表示大端字节序
  - 函数无返回值，只能单独使用或配合变量赋值

### 新增 / Added
- ✨ 新增 `bitmap()` 函数，支持参数化的字节序控制
- 📚 新增文档：`bitmap函数说明.md` 和 `bitmap-Function-Guide.md`

### 改进 / Improved
- 🎯 简化了位结构查看的语法，更符合函数式编程习惯
- 📖 更新了所有示例代码以使用新的 bitmap 函数语法
- 🔧 优化了代码结构，移除了全局字节序状态

### 移除 / Removed
- ❌ 移除了 `endian:` 命令（被 bitmap 函数参数替代）
- ❌ 移除了 `bitmap` 关键字语法（改为函数调用）
