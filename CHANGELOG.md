# 更新日志

## [1.0] - 2026-02-11

### 新增功能

#### 通用功能

- 版本号系统，从 1.0 开始
- 中英文双语支持
- 默认英文模式

#### GUI 界面

- 清空按钮增加快捷键 Ctrl+D
- 中英文切换按钮（中文/EN）
- 字体放大缩小功能（A+/A- 按钮）
- 快捷键 Ctrl+Plus 和 Ctrl+Minus 用于字体缩放
- 标题栏显示版本号
- 英文模式下所有界面元素（按钮、标签、示例）都使用英文
- 状态栏消息支持中英文
- 按钮栏支持水平滚动，防止字体放大后按钮消失
- 智能字体大小限制系统，根据屏幕尺寸自动计算最大字体
- 固定高度按钮容器，确保按钮始终可见
- 字体大小实时显示标签
- 默认英文界面

#### 命令行界面

- 中英文模式选择参数 `--lang` 或 `-l`
- 示例在 help 信息中显示，包含 bitmap 和 endian 关键字用法
- 标题显示版本号
- 英文模式下全英文显示
- `--version` 或 `-v` 参数查看版本号
- 默认英文模式

### 修复问题

- GUI 字体放大后按钮消失的问题（使用 grid 布局系统，固定组件比例）
- 状态栏重复显示信息的问题（移除递归调用，优化状态更新逻辑）
- 字体缩放时的无限递归问题
- 默认语言设置为英文
- help 信息增加 bitmap 和 endian 关键字的详细用法说明
- 语言切换逻辑优化，避免重新创建整个界面
- 按钮布局使用固定高度容器（60px），确保始终可见
- 状态栏使用固定高度（25px），只显示当前信息
- 输入/输出窗口占据固定比例，字体变化不影响窗口大小
- 改进滚动条显示逻辑，只在需要时显示
- 添加字体大小实时显示标签
- 状态栏消息自动恢复机制（3秒后恢复正常状态）
- 移除字体大小限制，最大可达 32

### 功能改进

- README.md 更新，包含新功能说明
- 代码结构优化，支持语言切换
- 更好的用户体验

### 技术细节

- Python 3.6+ 兼容
- 使用 argparse 处理命令行参数
- GUI 使用 tkinter 标准库
- 无需额外依赖

## 版本说明

版本号格式：主版本号.次版本号

- 主版本号：重大功能更新或不兼容的 API 变更
- 次版本号：向后兼容的功能性新增
