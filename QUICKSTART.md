# 快速入门

## 5分钟上手 CalcPaper

### 1. 运行程序

```bash
# GUI 版本（推荐新手）
python calc_paper_gui.py

# 命令行版本
python calc_paper.py
```

### 2. 基本使用

在输入区域输入：

```python
# 简单计算
a = 100
b = 200
和 = a + b
```

点击"计算"按钮（或按 F5），查看结果。

### 3. 位运算

```python
# 16进制运算
x = 0xFF
y = 0x0F
与 = x & y
```

### 4. 查看位结构

```python
# 设置字节序
endian: big

# 使用 bitmap 查看
bitmap 查看 = 0xFF
```

### 5. 快捷键

- F5：计算
- Ctrl+D：清空
- Ctrl+L：加载示例
- Ctrl+Plus：放大字体
- Ctrl+Minus：缩小字体

### 6. 切换语言

点击界面上的"EN"按钮切换到英文。

## 常见问题

### Q: 如何查看版本号？

```bash
python calc_paper.py --version
```

### Q: 如何切换语言？

**GUI**: 点击 "EN" 或 "中文" 按钮

**命令行**: 使用 `--lang` 参数

```bash
python calc_paper.py --lang zh
```

### Q: 如何保存结果？

**GUI**: 点击"保存结果"按钮或按 Ctrl+S

### Q: 支持哪些数值格式？

- 十进制: `123`, `456.78`
- 十六进制: `0xFF`, `0x1A2B`
- 二进制: `0b1010`, `0b11110000`
- 百分数: `15%`, `6.5%`

### Q: 如何查看详细的位结构？

使用 `bitmap` 关键字：

```python
endian: big
bitmap 查看 = 0xFF
```

### Q: 如何设置字节序？

```python
# 大端字节序
endian: big

# 小端字节序
endian: little
```

### Q: 程序支持哪些操作符？

**算术运算符**:
- `+` 加法
- `-` 减法
- `*` 乘法
- `/` 除法
- `%` 取模

**位运算符**:
- `&` 按位与
- `|` 按位或
- `^` 按位异或
- `<<` 左移
- `>>` 右移

## 示例演示

### 基本计算示例

```python
# 工资计算
工资 = 8000
房租 = 2000
餐费 = 30 * 25
结余 = 工资 - 房租 - 餐费
```

### 位运算示例

```python
# RGB颜色提取
endian: big
颜色 = 0xFF8040
红色 = (颜色 >> 16) & 0xFF
绿色 = (颜色 >> 8) & 0xFF
蓝色 = 颜色 & 0xFF

# 查看位结构
bitmap 查看颜色 = 颜色
```

### 百分数计算示例

```python
# 折扣计算
原价 = 100
折扣率 = 15%
折扣金额 = 原价 * 折扣率
实付 = 原价 - 折扣金额
```
