## 一、为什么要做这个工具？

### 痛点 1：进制转换太麻烦

在日常开发中，我们经常需要在十进制、十六进制、二进制之间转换。传统计算器要么不支持，要么操作繁琐。

**CalcPaper 的解决方案：**

- 直接在表达式中混用 `0xFF`、`0b1010`、`255`
- 自动识别并计算，无需手动转换

### 痛点 2：位运算结果难以理解

位运算的结果往往是一串数字，很难直观看出每一位的状态。

**CalcPaper 的解决方案：**

- 使用 `bitmap()` 函数，自动生成位结构可视化
- 支持大端/小端字节序显示
- 清晰展示每一位的索引和值

### 痛点 3：复杂计算需要多步骤

处理复杂的位运算时，往往需要多个中间变量，传统计算器无法保存这些变量。

**CalcPaper 的解决方案：**

- 支持变量定义和引用
- 像写代码一样写计算式
- 自动追踪变量依赖关系
- 🆕 输入变量名时自动弹出补全建议

### 痛点 4：日期计算靠数手指 🆕

项目排期、计算截止日期、推算工期……每次都要打开日历一天天数。

**CalcPaper 的解决方案：**

- 直接写日期表达式：`Y20260410 + D10` 就能算出 10 天后的日期
- 支持年月周天的加减运算
- 支持时间加减运算（时/分/秒）
- 两个日期相减直接得到天数差

## 二、核心功能展示

### 1. 多进制混合运算

```python
# 直接混用不同进制
a = 0xFF        # 十六进制
b = 0b1010      # 二进制
c = 255         # 十进制
sum = a + b + c # 自动计算
```

**输出：**

```
a = 0xFF                = 255
b = 0b1010              = 10
c = 255                 = 255
sum = a + b + c         = 520  # 255 + 10 + 255
```

### 2. 位运算可视化（bitmap 函数）

这是 CalcPaper 最强大的功能之一！

```python
# RGB 颜色提取
color = 0xFF8040
red = (color >> 16) & 0xFF
green = (color >> 8) & 0xFF
blue = color & 0xFF

# 使用 bitmap 查看位结构
bitmap(color, 1)
```

每一位的索引和值都清晰可见，调试位操作再也不用画图了！

### 3. 日期/时间运算 🆕

v2.0 新增的重磅功能！

```python
# 日期运算
today = Y20260410
deadline = today + D10      # Y20260420（10天后）
next_month = today + M1     # Y20260510（1个月后）
next_week = today + W2      # Y20260424（2周后）
diff = Y20260410 - Y20260101  # 99（天数差）

# 时间运算
start = T090000
end = T173000
duration = end - start      # 30600（秒数差）
lunch = start + h3 + m30    # T123000（3小时30分钟后）
```

**日期时长用大写**：`M`（月）、`W`（周）、`D`（天）

**时间时长用小写**：`h`（小时）、`m`（分钟）、`s`（秒）

项目排期、计算工期、推算截止日期，一行表达式搞定！

### 4. 变量名自动补全 🆕

在 GUI 中输入变量名时，自动弹出候选列表，Tab/Enter 确认选择。定义过的变量名不用再手动输入，减少拼写错误。

### 5. 字节序转换

```python
# 网络字节序（大端）转本地字节序（小端）
网络数据 = 0x12345678
本地数据 = swap(网络数据)  # 0x78563412

# 查看转换前后
bitmap(网络数据, 1)  # 大端显示
bitmap(本地数据, 0)  # 小端显示
```

### 6. 变量引用

```python
# 定义权限位
READ = 0b0001
WRITE = 0b0010
EXECUTE = 0b0100

# 组合权限
permission = READ | WRITE | EXECUTE
bitmap(permission, 1)

# 检查权限
has_read = permission & READ
has_delete = permission & 0b1000
```

像写代码一样自然！

### 7. 百分数计算

```python
# 购物折扣
price = 299
discount = 20%
coupon = 30
final_price = price * (1 - discount) - coupon
```

**输出：**

```
price = 299                                    = 299
discount = 20%                                 = 0.2
coupon = 30                                    = 30
final_price = price * (1 - discount) - coupon  = 209.2
```

## 四、实际应用场景

### 场景 1：嵌入式开发 - 寄存器配置

```python
# 配置 UART 控制寄存器
# [7:6] 波特率选择  [5:4] 数据位
# [3] 停止位  [2] 校验使能  [1:0] 模式选择

# 设置各字段
baud_rate = 0b10 << 6    # 115200
data_bits = 0b11 << 4    # 8位
stop_bit = 0b1 << 3      # 2个停止位
parity = 0b0 << 2        # 无校验
mode = 0b01              # 异步模式

# 组合配置
uart_config = baud_rate | data_bits | stop_bit | parity | mode

# 查看位结构
bitmap(uart_config, 1)
```

### 场景 2：网络编程 - IP 地址解析

```python
# IP: 192.168.0.1
IP = 0xC0A80001
bitmap(IP, 1)

# 提取各段
seg1 = (IP >> 24) & 0xFF  # 192
seg2 = (IP >> 16) & 0xFF  # 168
seg3 = (IP >> 8) & 0xFF   # 0
seg4 = IP & 0xFF          # 1
```

### 场景 3：项目排期 🆕

```python
# 项目排期计算
项目启动 = Y20260410
设计阶段 = 项目启动 + W2        # Y20260424（2周后）
开发阶段 = 设计阶段 + M1        # Y20260524（1个月后）
测试阶段 = 开发阶段 + W2        # Y20260607（2周后）
上线日期 = 测试阶段 + D3        # Y20260610（3天后）

# 总工期
总天数 = 上线日期 - 项目启动     # 61（天）
```

### 场景 4：工时计算 🆕

```python
# 今日工时
上班 = T090000
午休开始 = T120000
午休结束 = T133000
下班 = T180000

上午工时 = 午休开始 - 上班       # 10800（秒）= 3小时
下午工时 = 下班 - 午休结束       # 16200（秒）= 4.5小时
```

### 场景 5：权限管理

```python
# 文件权限系统
READ = 0b0001
WRITE = 0b0010
EXECUTE = 0b0100
DELETE = 0b1000

# 用户权限
user_perm = READ | WRITE | EXECUTE
bitmap(user_perm, 1)

# 管理员权限
admin_perm = READ | WRITE | EXECUTE | DELETE
bitmap(admin_perm, 1)

# 检查权限
can_delete = user_perm & DELETE  # 0，无删除权限
```
