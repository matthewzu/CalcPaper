# CalcPaper v2.0 Release Notes

## 🆕 新功能 / New Features

### 📅 日期/时间加减运算 / Date & Time Arithmetic
- 日期字面量 `Yyyyymmdd`（如 `Y20260410`）
- 时间字面量 `Thhmmss`（如 `T143000`）
- 日期时长（大写）：`Mxx`（月）、`Wxx`（周）、`Dxx`（天）
- 时间时长（小写）：`hxx`（小时）、`mxx`（分钟）、`sxx`（秒）
- 日期 ± 日期时长 → 日期
- 日期 - 日期 → 天数差
- 时间 ± 时间时长 → 时间
- 时间 - 时间 → 秒数差

### ✏️ 变量名自动补全 / Variable Name Autocomplete
- GUI 输入区域输入变量名时自动弹出候选列表
- Tab/Enter 确认选择，Escape 关闭，上下方向键导航

### 🔄 启动时自动检查更新 / Auto Update Check
- 启动后后台检查 GitHub 最新版本
- 发现新版本时在状态栏显示提示，点击可跳转下载

### 🖥️ GUI + CLI 双模式打包 / Dual-mode Packaging
- 统一入口脚本 `main.py`
- 默认启动 GUI，`--cli` 启动命令行模式
- `--lang zh/en` 指定语言，`--version` 查看版本

### 📦 版本号统一管理 / Unified Version Management
- `version.py` 作为版本号单一来源
- 所有模块从 `version.py` 导入 VERSION

## 🔒 保留关键字 / Reserved Keywords

以下前缀后跟数字的标识符为保留关键字，不可用作变量名：

| 前缀 | 含义 | 示例 |
|------|------|------|
| Y | 日期 | Y20260410 |
| T | 时间 | T143000 |
| M | 月（大写） | M3 |
| W | 周（大写） | W2 |
| D | 天（大写） | D10 |
| h | 小时（小写） | h2 |
| m | 分钟（小写） | m30 |
| s | 秒（小写） | s45 |

## ✅ 全部功能 / All Features

- 变量引用、多进制支持（16进制/2进制）
- 位运算（<<, >>, &, |, ^, ~）
- bitmap() 位结构可视化（大端/小端，宽度参数，前导0保留）
- swap() 字节序交换
- hex() 16进制显示
- 日期/时间加减运算
- 百分数计算
- 变量名自动补全
- 启动时自动检查更新
- GUI + CLI 双模式
- 会话自动恢复
- 配置持久化（窗口大小、语言、字体、快捷键、数据目录）
- 可自定义快捷键
- 中英文双语
- 字体缩放
- 撤销/恢复
- 智能注释（位运算自动显示16进制格式）
